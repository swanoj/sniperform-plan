Refer a Friend Campaign

Turn every customer into a brand ambassador — reward referrals with free merch & free shipping

The Offer

Refer a Friend to Sniperform

Share your unique link. When your friend makes their first purchase, you both win.

YOU GET

Free Sniperform T-Shirt
Valued at $50 — shipped free

YOUR FRIEND GETS

Free Shipping
On their entire first order
Why this works: The referrer reward ($50 t-shirt) feels premium and tangible — it's not a generic discount code people ignore. The friend gets free shipping which removes the #1 cart abandonment reason. Both sides have a clear, simple incentive.

How It Works

Customer buys → Gets referral email → Shares unique link → Friend orders with free shipping → Referrer gets free t-shirt
1

Customer completes a purchase

Any order on sniperform.com triggers the referral email flow (sent 3 days after delivery).

2

Referral email lands in their inbox

Personalised email with their unique referral link and a clear explanation of what they (and their friend) get.

3

Customer shares the link

They share via text, social, email — whatever works. The link auto-applies the friend's free shipping discount at checkout.

4

Friend places their first order

Free shipping is applied automatically. The referral app tracks the conversion back to the original customer.

5

Referrer gets their free t-shirt

Automatic notification sent to the referrer. T-shirt is either auto-fulfilled as a $0 order or a discount code for 100% off the tee is sent to them.

The maths: Even at a $28 all-in cost per referral, you're acquiring customers at roughly half the cost of paid ads — and referred customers have 25–40% higher LTV on average. The t-shirt also doubles as walking brand advertising.
T-shirt tip: Price the Sniperform tee at $50 on the store (it's a premium branded item). The perceived value to the referrer is $50, but your true cost is $12–18. This is what makes the economics work — high perceived value, low actual cost.

Segmentation

  • All purchasers — every customer who has completed at least one order
  • Existing customers (backfill) — one-time blast to all past purchasers to launch the programme
  • Repeat customers — higher priority, more likely to refer (consider sending these first)
  • Exclude — anyone who has already received a referral reward (they stay in the programme but don't get the initial invite again)

Shopify Setup — Step by Step

Option A — Referral App (Recommended)

Recommended approach: Use a dedicated referral app. It handles unique links, tracking, reward fulfilment, and fraud prevention out of the box.

Best Shopify Referral Apps:

AppPriceWhy
ReferralCandyFrom $59/moMost popular, solid tracking, auto-rewards, email templates, integrates with Klaviyo
Referral FactoryFrom $95/moMore customisable, landing page builder, good for branded experiences
Yotpo Loyalty & ReferralsFree tier availableIf you already use Yotpo for reviews — combines loyalty + referral in one
Smile.ioFree tier availableLoyalty + referral combo, great for points-based systems (can also do flat rewards)

Setup Steps — ReferralCandy (example)

  1. Shopify Admin → Apps → Search "ReferralCandy" → Install
  2. Configure the Referrer Reward:
    • Reward type: Custom Reward (or Discount Code)
    • Set up a 100% discount code auto-generated for the Sniperform T-Shirt product only
    • Alternatively: set reward to "Cash/Gift" at $50 value applied as store credit
  3. Configure the Friend Incentive:
    • Reward type: Free Shipping
    • Auto-generates a unique free shipping code for each referral link
  4. Customise the referral landing page with Sniperform branding
  5. Set up email templates (use the copy above as a starting point)
  6. Connect to Klaviyo (if using) for email automation
  7. Test the full flow before launching

Option B — Manual Setup (No App, DIY)

Use this if you want zero app costs. More manual work but fully doable. Best for smaller volume.

Step 1 — Create the T-Shirt Product

Shopify Admin → Products → Add product

  • Title: Sniperform T-Shirt
  • Price: $50.00 (so it's purchasable by anyone too)
  • Add size variants (S, M, L, XL, XXL)
  • Upload product images
  • Track inventory so you know stock levels

Step 2 — Create Discount Codes

Shopify Admin → Discounts → Create discount

Referrer Reward Code:

  • Code: SNIPERFORM-FREETEE (or generate unique codes per customer)
  • Type: Percentage — 100%
  • Applies to: Specific product — Sniperform T-Shirt only
  • Usage: One per customer
  • Also tick: Free shipping on this order

Friend Free Shipping Code:

  • Code: FRIENDSHIP (or unique per referral)
  • Type: Free shipping
  • Minimum requirement: No minimum (or set a minimum order if preferred)
  • Usage: One per customer
  • Set an end date if running as a limited campaign

Step 3 — Set Up Email Automation

Shopify Admin → Marketing → Automations → Create automation

Or use Klaviyo / Shopify Email:

  1. Trigger: Order fulfilled (delivered)
  2. Wait: 3 days
  3. Send: Referral invite email (with unique discount code for friend)
  4. Condition: If no referral conversion after 10 days → send reminder email
  5. Trigger: When friend's order is placed with the referral code → send reward email to referrer with free t-shirt code

Step 4 — Track Referrals Manually

If not using an app, track referrals in a spreadsheet or Shopify order tags:

  • Tag referred orders with referral and the referrer's email
  • Shopify Admin → Orders → Filter by tag "referral"
  • When a referral order comes in, manually send the free t-shirt code to the referrer
  • Or use Shopify Flow (Shopify Plus / Advanced) to automate tagging and notifications

Option C — Shopify Flow Automation (Advanced/Plus plans)

Automated Flow

Shopify Admin → Apps → Shopify Flow → Create workflow

  1. Trigger: Order created
  2. Condition: Discount code used contains "FRIEND-" prefix
  3. Action: Tag order as "referral"
  4. Action: Look up the referrer (mapped from the discount code)
  5. Action: Send internal notification to fulfil free t-shirt
  6. Action: Send email to referrer with reward claim link

Fraud Prevention

Watch for:
  • Self-referrals (same person, different email) — referral apps detect this automatically; if manual, check shipping addresses
  • Referral + immediate return — set a rule: t-shirt reward only ships after the friend's return window closes (14–30 days)
  • Bulk fake referrals — cap rewards at 5–10 per customer per month initially
